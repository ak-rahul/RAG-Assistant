# Changelog

All notable changes to RAG-Assistant will be documented here.

## [1.0.0] - 2024-12-17

### Added
- Streamlit web interface
- CLI with multiple commands
- FastAPI server
- Support for PDF, DOCX, TXT, MD, JSON
- ChromaDB vector storage
- Groq and OpenAI LLM integration
- Conversation memory
- Source citation
- Configurable via YAML

- Troubleshooting guide
- Sample documents
